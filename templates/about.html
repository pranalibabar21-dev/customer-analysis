About Us

Customer Sales & Churn Analysis System is a web-based application developed to analyze customer sales data, predict future sales trends, identify customer churn patterns, and perform customer segmentation using Machine Learning techniques.


Project Objectives

  • Analyze customer sales performance
  • Predict future sales using Machine Learning
  • Identify customer churn trends
  • Perform customer segmentation
  • Generate business insights and reports

Technologies Used

Python, Flask, SQLite, Pandas, Matplotlib, Scikit-Learn and Bootstrap.


Key Features

  • User Registration & Login
  • Admin Dashboard
  • CSV Dataset Upload
  • Sales Prediction
  • Customer Segmentation
  • Customer Churn Analysis
  • PDF Report Generation
  • Excel Export
